# La Bodega del Barrio — Drinks Shop Web App

A simple Node.js + Express + EJS web application for a fictional drinks shop business. The app displays shop information, product categories, shop locations, manager details, and includes a customer registration and survey form.

---

## Features

### Core Pages
- **Home Page** – Displays the shop name and available drink categories.
- **About Page** – Shows details about the business, including shop locations, shop managers, and addresses.
- **Register Page** – A form that collects user first name, surname, and email (with validation).
- **Search Page** – Allows users to search for product categories.
- **Survey Page** – Customers can fill out a survey with:
  - First name
  - Surname
  - Email
  - Age
  - Preferred drink category (radio buttons)
  - Student status (checkbox)
- **Survey Results Page** – Displays the user's submitted survey responses.

---

## Technologies Used
- **Node.js**
- **Express.js** (for routing and server setup)
- **EJS** (templating engine)
- **CSS** served through Express static files
